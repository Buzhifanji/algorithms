## 介绍

插入排序是一种对**少量元素**排序非常有效的排序算法，在日常生活中常见的一个例子是打扑克牌时插入扑克牌，为了清楚知道自己拥有什么牌，以及方便自己快速抽取想要的牌，我们在一开始拿到牌的时候，会习惯性把牌插入到已排序的扑克牌中。

![alt](/ins-sort.gif)

## 伪代码

```console
for j = 2 to A.length
    key = A[j]
    // Insert A[j] into the sorted sequence A[1..j - 1]
    i = j - 1
    while i > 0 && A[i] > key
        A[i + 1] = A[i]
        i = i - 1
    A[i + 1]  = key
```
